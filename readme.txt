== Description == 
This plugin adds a shortcode to add a custom map from a simple address. using The latest Google API. 
 
== Installation == 
1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation.
1. Then activate the Plugin from Plugins page.
1. And Last add the shortcode anywhere you want
